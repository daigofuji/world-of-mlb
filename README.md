# world-of-mlb
2018 MLB Roster by birth countries


Build using [daigofuji/jekyll-foundation-6-starter](http://daigofuji.github.io/jekyll-foundation-6-starter/)

For Dev:
```
npm start
bundle exec jekyll serve --watch
```

(Run this once:
  `npm install` and `bundle install`)
